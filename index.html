Tango Core Classes Reference Documentation

Introduction

This is the reference documentation for all classes provided by the Tango C++ API. These classes are divided in two groups (modules) which are

  • Client classes
  • Server classes

Client classes are mostly used in application(s) acting as clients and dealing with Tango devices. Server classes are mostly used in Tango class or device server process main function.

In order to develop Tango's related software, it's a good idea to have a look at the Tango Controls documentation, especially the Developer's Guide section.

A look at the so-called "Related Pages" could also help the developer.

Useful information is also available on the Tango Web site.
